# Arithmetic API

## /add

Returns the sum of two numbers

```bash
curl http://localhost:8000/add?a=1&b=2
# 3
```

## /subtract

Returns the difference between two numbers

```bash
curl http://localhost:8000/subtract?a=5&b=3
# 2
```
